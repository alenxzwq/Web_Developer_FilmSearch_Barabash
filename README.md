   Movie Search — это веб-приложение для поиска информации о фильмах и сериалах. Проект разработан в учебных целях и демонстрирует навыки работы с API, построение         динамических интерфейсов, реализацию пагинации, фильтрации и адаптивного дизайна.

   Приложение позволяет пользователям просматривать популярные фильмы и сериалы, искать контент по ключевым словам, применять фильтры, читать отзывы и смотреть трейлеры.


    Основной функционал
        1. Просмотр популярных фильмов и сериалов
              -На странице "Фильмы" отображается сетка из популярных фильмов
              -На странице "Сериалы" отображается сетка из популярных сериалов
              -На мобильных устройствах фильмы/сериалы отображаются в виде сериала

        2. Пагинация
              -На одной странице отображается 9 карточек
              -Если фильмов больше 9, появляются кнопки с номерами страниц
              -При клике на номер страницы — отображаются следующие 9 фильмов
              -Карточки сортируются по рейтингу (от большего к меньшему)

        3. Поиск
              -Поиск работает на всех страницах
              -При вводе запроса и нажатии кнопки "Найти" пользователь перенаправляется на страницу результатов
              -Результаты поиска отображаются с пагинацией (9 карточек на страницу)
              -На странице фильмов ищутся только фильмы, на странице сериалов — только сериалы
              -Поиск с главной страницы показывает и фильмы, и сериалы

        4. Фильтрация
              -На странице списка фильмов/сериалов доступны фильтры: год выпуска, жанр, страна, рейтинг
              -Фильтры применяются автоматически при изменении любого параметра
              -Можно комбинировать несколько фильтров одновременно
              -Если ничего не найдено — выводится соответствующее сообщение

        5. Слайдер 
              -Горизонтальная прокрутка карточек
              -Кнопки "Назад" и "Вперед" для навигации
              -Touch-свайпы на мобильных устройствах
              -Кнопки отключаются в начале и конце слайдера

        7. Тёмная тема
              -Переключение между светлой и тёмной темами
              -Кнопка переключения есть в десктопном меню и в мобильном боковом меню
              -Выбранная тема сохраняется в localStorage
              -При первом визите определяется системная тема пользователя

        8. Бургер-меню
              -На маленьких экранах стандартное меню скрывается, появляется бургер-иконка
              -При клике на бургер открывается боковое меню
              -Закрытие происходит по крестику или клику на оверлей
              -При открытии меню блокируется скролл страницы
